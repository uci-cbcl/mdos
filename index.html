MDOS: a method for Motif Discovery using Orthologous Sequences

MDOS is a comparative genomics method for regulatory motif discovery. Different from previous methods, it does not require or use sequence alignments. It is particularly suitable for discovering motifs in highly diverged genomes, or genomes with sequence alignments difficult to obtain or poor in quality.

The code is written in Java and can be downloaded from here. You will need Java version 1.5 or above to run the code.
